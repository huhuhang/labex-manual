# 技能树

## 什么是技能树

技能树是串联 LabEx 内容资源的骨架，它将不同的 Challenges 和 Labs 通过技能点系统化关联起来。

![Skills Tree](.github/assets/skill-tree.png)

技能树一般由 LabEx 官方根据技术方向的技能点确定，我们会及时更新技能树。

如果作者希望为 Challenges 和 Labs 增加新的技能点，需要向 LabEx 申请，我们会审核后添加到系统中。

> 查看最新的技能树 [**↗ Skills Tree**](https://labex-dev.feishu.cn/base/bascnNz4Nqjqgqm1Nm5AYke6xxb).

## 关联到步骤

技能点会通过 `index.json` 配置文件关联到步骤中，我们会为每个技能点分配唯一的 `SKILL ID`，你需要从 [最新的技能树](https://labex-dev.feishu.cn/base/bascnNz4Nqjqgqm1Nm5AYke6xxb) 中复制粘贴 `SKILL ID`。

![The Structure of Labs and Challenges](.github/assets/basic-resource-structure.png)

```json copy
"steps": [
    {
    "title": "Bananas",
    "text": "step1.md",
    "verify": [],
    "skills": ["linux/ls", "linux/cd"] // 技能点 SKILL ID 列表，主要技能点放前面
}]
```

import { Callout } from "nextra-theme-docs";
